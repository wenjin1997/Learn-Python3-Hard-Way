# 习题1-5
## 习题1 第一个程序
* 学习了打印print的用法。
* 单引号也是可以打印句子的。
* 如果要打印的句子中有单引号，那么外面要用双引号引起来。代码`print('I'd much rather you 'not'.'')`会报语法错误。
* 如果要打印的句子中有双引号，那么外面要用单引号引起来。代码`print("I "said" do not touch this.")`是会报语法错误。
* 如果要在打印的一行中既有单引号又有双引号怎么办？我现在的思路是可以分别打印，然后再去掉换行符。补充，可以使用格式化字符串或者.format()语法。
* 我用的文本编辑器是Visual Studio Code，那么在运行之前一定要先保存，不然运行的是没保存之前的代码！

## 习题2 注释和#号
* 学习了如何注释进行注释，在一行的开头用#号。
* 如果要进行多行注释，在每一行前面加上#。
* 倒着读代码是检查代码是否错误的一个好的方法。

## 习题3 数字和数学计算
* 学习了基本的数学计算用到的符号。
* 注意%是取余的意思。
* 修改代码为`print("Is it greater?", 5 > --2)`，也能运行。会有个错误提醒"Use of the non-existent -- operator"。

## 习题4 变量和命名
* 回顾了变量与命名，适当的变量名能更加容易记住程序内容，也方便阅读。
* space_in_a_car这样的命名方式很好，可以把下划线看作是空格，便于代码阅读与检查。

## 习题5 更多的变量和打印
* “格式化字符串（format string）” 可以将更多的变量方便的打印出来。
* 在字符串中嵌入变量，将变量放在花括号里，在前面加上"f"(format)，例如f"Hello {somevar}"。
* 字符串的作用：打印、存入文件、发送给Web服务器等。
* 浮点数的四舍五入用round()函数，如round(1.73333)。
